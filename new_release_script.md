# Add New Release Script

## Overview
This script is designed to automate the process of fetching updates, rebasing, and rebuilding the project with ease. It aims to streamline the release management workflow, ultimately reducing manual intervention and potential errors.

## Features
- Automatically fetches updates from the remote repository.
- Performs a rebase on the current branch.
- Initiates build processes seamlessly.

## Usage
Run this script before starting your release process to ensure your branch is up-to-date and rebuilt with the latest code.
